# 键来

> 大千世界，无奇不有。
> 
> 我陈平安，唯有一键，可搬山，倒海，降妖，镇魔，敕神，摘星，断江，摧城，开天！
> 
> 我叫陈平安，平平安安的平安。我是一名键客。

---

本秘籍旨在成为最好懂的键仙修行秘籍，实现天下人人皆可修行的宏愿。

## 构建

```shell
mdbook build
```

## 预览

```shell
mdbook serve
```
